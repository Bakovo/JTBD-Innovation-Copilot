# Innovation Classification Logic

## Decision Tree

IS the job currently served by existing solutions?
  YES -> Can we serve it significantly better?
    YES -> Incremental (or Breakthrough if new tech)
    NO  -> Business Model or Service
  NO -> Is the barrier cost/complexity?
    YES -> Disruptive (serve non-consumption)
    NO  -> Breakthrough (new outcome)

## Classification Indicators

| Type | Users | Solution | Market | Risk |
|------|-------|----------|--------|------|
| Incremental | Existing | Better | Known | Low |
| Breakthrough | Existing/New | Novel | Uncertain | High |
| Disruptive | Non-consumers | Simpler | New | Medium |
| Business Model | Existing | Same | Same | Medium |
| Service | Existing | Augmented | Adjacent | Low-Med |
