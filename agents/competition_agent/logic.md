# Competition Analysis Logic

## Analysis Order

1. Start with switch data (if available) -> current and previous solutions
2. Ask: what were users doing BEFORE the current solution?
3. Ask: what do users do when the current solution FAILS?
4. Ask: who CANNOT use current solutions? Why?

## Non-consumption Detection

Signals of non-consumption:
- "I wish I could but..."
- "It's too expensive/time-consuming/complicated"
- "I just put up with it"
- "I asked someone else to do it"
